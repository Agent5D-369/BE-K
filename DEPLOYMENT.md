# Deployment

## GitHub Pages
1. Create or open your GitHub repository.
2. Upload the full contents of this folder.
3. In repository settings, enable GitHub Pages from the root or `/docs` depending on your preferred structure.
4. Confirm that `index.html` is at the published root.
5. Add your custom domain if desired.
6. Update the canonical URL, OG URL, robots.txt, sitemap.xml, and structured data to match your live domain.

## Required replacements
- Homepage canonical URL
- Open Graph image URL
- Hello Audio checkout URL
- Draft legal copy
- Any optional analytics or email form endpoint
