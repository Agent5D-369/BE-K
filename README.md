# Bridging Earth and Kanaria landing page package

This is a static, GitHub Pages-friendly landing page package built around a free diagnostic quiz and a primary audiobook CTA.

## What is included
- Premium single-page landing page
- Self-contained interactive quiz
- Clean CTA handoff to the audiobook offer
- Draft legal pages
- SEO, metadata, sitemap, robots, and llms.txt
- Strategy, analytics, image planning, and QA docs

## Before launch
1. Replace every `https://example.com/` placeholder with your real domain.
2. Replace the Hello Audio button placeholder with your real checkout URL.
3. Add your real Open Graph image assets.
4. Replace draft legal pages with reviewed final versions.
5. Add analytics only after deciding your stack.
6. If you want email capture, connect a static form provider or email platform.

## Hosting
This package is designed for static hosting such as GitHub Pages.

## Main conversion logic
Free quiz first.
Recognition and share loop second.
Audiobook purchase third.
